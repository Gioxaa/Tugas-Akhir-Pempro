# Aplikasi Manajemen Kost Simple

Program untuk mengelola data kost sederhana. Dibuat untuk tugas Pengenalan Pemrograman semester 1.

## Feature
- Tambah penghuni baru
- Lihat semua kamar
- Edit data kamar
- Kosongkan kamar
- Cari penghuni

## How to Use

**Compile:**
```bash
gcc *.c -o main
```

**Run:**
```bash
./main
```

Pilih menu 1-6 sesuai kebutuhan.
