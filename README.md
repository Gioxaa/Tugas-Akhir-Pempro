# Aplikasi Manajemen Kost

Program untuk mengelola data kost sederhana. Dibuat untuk tugas Pengenalan Pemrograman semester 1.

## Fitur
- Tambah penghuni baru
- Lihat semua kamar
- Edit data kamar
- Kosongkan kamar
- Cari penghuni

## Cara Menggunakan

**Compile:**
```bash
gcc main.c tambahPenghuni.c tampilkanKamar.c editKamar.c kosongkanKamar.c cariPenghuni.c keluar.c -o main
```

**Jalankan:**
```bash
./main
```

Pilih menu 1-6 sesuai kebutuhan.
